HueSearch 幫助設計師透過選擇主色快速找到圖片，輕鬆建立 Mood Board、或用於分類檔案的 thumbnails 等
HueSearch helps designers quickly find images by selecting a primary color, making it easy to create mood boards or use as thumbnails for organizing files.


## Environment Variable

This project uses Unsplash API.

For demo purposes, the Access Key is defined directly in JavaScript.
In production, this should be moved to a backend or build-time environment variable.
